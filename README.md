# e-plantShopping 🌿

## Project Overview
e-plantShopping is a React-based plant shop application that allows users to browse plants, add them to a cart, and manage their shopping experience.

## Features
- Landing page with "Get Started"
- Product listing with categories
- Add to Cart functionality (Redux)
- Cart page with quantity management
- Dynamic total price calculation

## Technologies Used
- React.js
- Redux Toolkit
- React Router
- CSS

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
